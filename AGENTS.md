# AGENTS.md

## Cursor Cloud specific instructions

- This repo is a single-file Python project (`hello.py`) used for Cursor Agent × Linear integration testing. There are no third-party dependencies, no build step, and no test/lint tooling.
- System `python3` (3.12+) is preinstalled; no virtualenv or package install is required.
- Run the app with `python3 hello.py` (see `README.md`). Expected output: `Hello, World! Greetings from Cursor Agent × Linear integration.`
